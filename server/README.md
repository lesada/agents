# 🛡️ Agents Server

This is the Fastify backend API for the Agents project.

## 🛠️ Setup

```sh
npm install
npm run dev
```

- Runs on http://localhost:3333 by default

## 📁 Structure

- `src/` – API source code
- `.env` – Environment variables

## 📝 Notes

- Uses [Fastify](https://fastify.dev/) + [Zod](https://zod.dev/)
- See `biome.json` for code style
